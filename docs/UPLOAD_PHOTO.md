# BotFather 사진 업로드 방법

## 📸 간단한 사진 업로드 방법

BotFather가 사진을 필수로 요구하므로, 간단한 이미지를 업로드해야 합니다.

---

## 방법 1: 온라인 이미지 생성기 사용 (가장 쉬움)

### 1단계: 이미지 생성
1. 브라우저에서 검색: **"placeholder image 640x360"**
2. 또는 이 URL 직접 열기: `https://via.placeholder.com/640x360`
3. 이미지가 나타나면 **우클릭 → 이미지 저장**

### 2단계: Telegram에서 업로드
1. BotFather 채팅창에서 **📎 (클립) 아이콘** 클릭
2. **Photo** 선택
3. 저장한 이미지 선택
4. **Send** 클릭

---

## 방법 2: 간단한 로고 이미지 만들기

### 온라인 도구 사용:
1. 브라우저에서 검색: **"Canva 640x360"** 또는 **"Figma 640x360"**
2. 간단한 텍스트 이미지 만들기:
   - 배경색: 어두운 색 (예: #0A0A0F)
   - 텍스트: "KISMET" (큰 글씨, 금색 또는 흰색)
   - 크기: 640x360 픽셀로 저장
3. 저장 후 Telegram에서 업로드

---

## 방법 3: 기존 이미지 사용 (가장 빠름)

### 프로젝트에 이미지가 있다면:
1. 프로젝트 폴더에서 적절한 이미지 찾기
2. 이미지 편집 프로그램으로 640x360으로 리사이즈
3. Telegram에서 업로드

---

## 방법 4: 간단한 색상 이미지 (초간단)

### Paint나 간단한 편집 프로그램 사용:
1. 새 이미지 만들기: 640x360 픽셀
2. 배경색 채우기 (예: 어두운 보라색 #0A0A0F)
3. 저장 (PNG 또는 JPG)
4. Telegram에서 업로드

---

## ⚡ 가장 빠른 방법 (추천)

1. 브라우저에서 이 URL 열기:
   ```
   https://via.placeholder.com/640x360/0A0A0F/FFFFFF?text=KISMET
   ```
   - 어두운 배경에 흰색 텍스트 "KISMET" 이미지가 생성됩니다

2. 이미지 **우클릭 → 이미지 저장**

3. Telegram BotFather 채팅창에서:
   - **📎 (클립) 아이콘** 클릭
   - **Photo** 선택
   - 저장한 이미지 선택
   - **Send** 클릭

---

## ✅ 다음 단계

사진 업로드 후:
1. BotFather가 **Web App URL**을 물어볼 것입니다
2. `https://kismet-beta.vercel.app` 입력
3. 완료!

---
