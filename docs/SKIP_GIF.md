# BotFather GIF 건너뛰기

## ✅ GIF는 선택사항입니다!

BotFather가 GIF를 요청하지만, **`/empty` 명령어로 건너뛸 수 있습니다**.

---

## 📋 방법: `/empty` 명령어 사용

### 1단계: 건너뛰기
BotFather 채팅창에 다음 입력:
```
/empty
```

### 2단계: 다음 단계로 진행
- BotFather가 **Web App URL**을 물어볼 것입니다
- `https://kismet-beta.vercel.app` 입력

---

## ✅ 다음 단계

`/empty` 입력 후:
1. BotFather가 **Web App URL**을 물어볼 것입니다
2. `https://kismet-beta.vercel.app` 입력
   - ⚠️ `https://` 포함
   - 끝에 `/` 없이 입력
3. 완료!

---

## 💡 참고

- GIF는 나중에 `/editapp`으로 추가할 수 있습니다
- 지금은 Web App URL 설정이 가장 중요합니다
- GIF 없이도 Mini App은 정상 작동합니다

---
