# BotFather 사진 업로드 건너뛰기

## ✅ 사진은 선택사항입니다!

BotFather가 사진을 요청하지만, **건너뛸 수 있습니다**.

---

## 📋 방법 1: 건너뛰기 (권장)

### 1단계: 사진 없이 진행
1. BotFather에게 **"Skip"** 또는 **"건너뛰기"** 입력
2. 또는 그냥 **다음 단계로 넘어가기** (Web App URL 입력)

### 2단계: Web App URL 입력
BotFather가 다음으로 물어볼 것입니다:
- **Web App URL**: `https://kismet-beta.vercel.app` 입력

---

## 📋 방법 2: 간단한 사진 만들기 (선택사항)

만약 사진을 넣고 싶다면:

### 1단계: 온라인 이미지 생성기 사용
1. 브라우저에서 검색: "640x360 image generator"
2. 또는 간단한 로고/텍스트 이미지 만들기
3. 640x360 픽셀로 저장

### 2단계: 업로드
1. Telegram 채팅창에서 **📎 (클립) 아이콘** 클릭
2. **Photo** 선택
3. 만든 사진 업로드

---

## ⚠️ 중요: Web App URL이 더 중요합니다!

사진은 나중에 추가해도 됩니다. **지금은 Web App URL을 입력하는 것이 중요합니다.**

---

## 📝 다음 단계

1. BotFather에게 **"Skip"** 또는 그냥 **Web App URL 입력**
2. **Web App URL**: `https://kismet-beta.vercel.app` 입력
3. 완료!

---

## 💡 팁

- 사진 없이도 Mini App은 정상 작동합니다
- 나중에 `/editapp` 명령어로 사진을 추가할 수 있습니다
- 지금은 URL 설정이 우선입니다
