# 사진 업로드 간단 방법

## 🎨 방법 1: 온라인 이미지 검색 (가장 쉬움)

### 1단계: 구글 이미지 검색
1. 브라우저에서 구글 열기
2. 검색: **"640x360 image"** 또는 **"640x360 banner"**
3. 아무 이미지나 하나 선택
4. 이미지 **우클릭 → 이미지 저장**

### 2단계: Telegram에서 업로드
1. BotFather 채팅창에서 **📎 (클립) 아이콘** 클릭
2. **Photo** 선택
3. 저장한 이미지 선택
4. **Send** 클릭

---

## 🎨 방법 2: 간단한 색상 이미지 만들기

### Windows Paint 사용:
1. **Windows 키** 누르기
2. **"Paint"** 검색 → 실행
3. **파일 → 속성** 클릭
4. 너비: **640**, 높이: **360** 입력
5. **확인** 클릭
6. 아무 색상으로 채우기 (예: 검은색, 보라색)
7. **파일 → 저장** → 이름: `kismet.png` → 저장
8. Telegram에서 업로드

---

## 🎨 방법 3: 스크린샷 사용 (초간단)

### 1단계: 간단한 화면 만들기
1. 브라우저에서 아무 웹페이지나 열기
2. **F12** 눌러서 개발자 도구 열기 (선택사항)
3. 또는 그냥 아무 화면이나 캡처

### 2단계: 이미지 편집
1. **Windows 키 + Shift + S** 눌러서 캡처 도구 열기
2. 영역 선택 (가로로 길게)
3. Paint에 붙여넣기 (Ctrl + V)
4. 크기 조정: **이미지 → 크기 조정**
5. 픽셀 선택 → 너비: **640**, 높이: **360**
6. 저장
7. Telegram에서 업로드

---

## 🎨 방법 4: 온라인 이미지 에디터 사용

### Canva 사용 (무료):
1. 브라우저에서 [canva.com](https://canva.com) 열기
2. **무료로 시작** 클릭
3. **사용자 지정 크기** 클릭
4. 너비: **640**, 높이: **360** 입력
5. 간단한 디자인 만들기 (텍스트 "KISMET" 추가)
6. **다운로드** 클릭
7. Telegram에서 업로드

---

## ⚡ 가장 빠른 방법 (추천)

### 간단한 색상 이미지:
1. **Paint** 열기
2. 새 파일 만들기
3. 크기: **640 x 360**
4. 어두운 색으로 채우기
5. 저장
6. Telegram에서 업로드

---
